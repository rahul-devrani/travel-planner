# Explore DevBhoomi – Smart Travel Planner for Uttarakhand

## Project Overview
**Explore DevBhoomi** is a web-based travel planning platform designed for travelers who want to explore the beauty of Uttarakhand. The platform provides a centralized solution where users can discover destinations, receive personalized recommendations, estimate travel budgets, and plan trips efficiently.

---

##  Objectives
- Develop a web-based tourism planner for Uttarakhand.
- Provide detailed information about tourist destinations.
- Suggest destinations based on user interests and travel preferences.
- Estimate trip budgets dynamically.
- Help users plan complete trips in one place.

---

## Features
-  **Homepage** – User-friendly landing page with navigation.
-  **Destination Explorer** – Browse popular places in Uttarakhand.
-  **Smart Recommendations** – Suggest destinations based on user interests.
-  **Budget Calculator** – Estimate travel costs based on inputs.
-  **User Authentication** – Secure registration and login system.
-  **Reviews & Feedback** – Users can share experiences.
-  **Weather Forecast** – Real-time weather updates using API.
-  **Interactive Maps** – Location visualization using OpenStreetMap.

---

##  System Architecture

The application follows a **Client-Server Architecture**:

###  Frontend
- Built using **HTML, CSS, JavaScript**
- Handles user interaction and UI rendering
- Communicates with backend via APIs

###  Backend
- Developed using **Node.js** and **Express.js**
- Handles:
  - User authentication
  - Recommendation logic
  - Budget calculation
  - API integrations

###  Database
- **MySQL** is used for structured data storage:
  - User data
  - Destinations
  - Tourist places
  - Reviews

---

##  APIs & Tools Used
-  **OpenWeatherMap API** – Real-time weather data
-  **Leaflet.js + OpenStreetMap** – Interactive maps and geolocation
-  REST APIs – Communication between frontend and backend
